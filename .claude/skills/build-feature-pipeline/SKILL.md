# PURPOSE

Build modular feature engineering pipelines
for ML, telemetry, and industrial predictive workflows.

The skill should structure reusable,
maintainable, and scalable feature workflows.

---

# RESPONSIBILITIES

- Build feature engineering workflows
- Coordinate temporal feature generation
- Coordinate rolling-window processing
- Structure feature dependency chains
- Improve feature maintainability
- Support predictive maintenance workflows
- Support scalable feature generation

---

# NON-GOALS

- Do not generate uncontrolled feature explosion
- Do not fabricate feature requirements
- Do not ignore temporal leakage risks
- Do not tightly couple feature stages

---

# TOOLS

- Read
- Write

---

# INPUTS

- Dataset structure
- Feature requirements
- ML objectives
- Operational constraints (optional)

---

# WORKFLOW

1. Analyze feature requirements
2. Identify feature dependencies
3. Build modular feature stages
4. Validate temporal integrity
5. Optimize scalability and maintainability
6. Produce structured feature pipeline

---

# OUTPUT FORMAT

# Feature Pipeline Overview
# Feature Stages
# Dependency Structure
# Leakage Risks
# Scalability Considerations
# Recommendations
# Confidence Level

---

# QUALITY BAR

- Prioritize modularity
- Preserve temporal integrity
- Avoid feature explosion
- Focus on maintainability and scalability
- Preserve predictive usefulness

---

# FAILURE BEHAVIOR

If feature requirements are incomplete:
- explicitly state uncertainty
- avoid unsupported feature assumptions
- build only evidence-supported workflows
