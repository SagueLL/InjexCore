name: decompose-objective
description: Decompose complex engineering, ML, industrial, or operational objectives into structured executable subtasks — surfacing logical breakdown, dependencies, parallelization opportunities, and execution priorities, without fabricating requirements. Use when the user asks to plan, break down, decompose, scope, or structure a complex task or multi-step workflow.
---

# PURPOSE

Decompose complex engineering, ML, industrial,
or operational objectives into structured executable subtasks.

The skill should improve workflow clarity,
coordination, and execution efficiency.

---

# RESPONSIBILITIES

- Break down complex objectives
- Identify logical subtasks
- Detect workflow dependencies
- Identify parallelizable work
- Reduce execution ambiguity
- Improve workflow structure
- Support multi-agent coordination

---

# NON-GOALS

- Do not fabricate requirements
- Do not overcomplicate simple objectives
- Do not generate unnecessary subtasks
- Do not assume hidden constraints

---

# TOOLS

- Read
- Write

---

# INPUTS

- High-level objective
- Optional constraints
- Optional ecosystem context

---

# WORKFLOW

1. Analyze the objective
2. Identify major workflow components
3. Detect dependencies and sequencing
4. Decompose into executable subtasks
5. Prioritize and structure execution
6. Produce structured workflow breakdown

---

# OUTPUT FORMAT

# Objective Summary
# Workflow Breakdown
# Task Dependencies
# Parallelization Opportunities
# Execution Priorities
# Risks and Unknowns
# Recommendations
# Confidence Level

---

# QUALITY BAR

- Prioritize clarity and maintainability
- Avoid unnecessary complexity
- Preserve logical workflow structure
- Focus on execution usefulness
- Preserve contextual awareness

---

# FAILURE BEHAVIOR

If objectives are ambiguous:
- explicitly state uncertainty
- avoid unsupported decomposition
- identify missing context

---

# FUTURE EXTENSIONS

Possible future capabilities:
- Dynamic workflow generation
- Autonomous task prioritization
- Recursive workflow decomposition
- Cross-agent optimization
