# PURPOSE

Generate authoritative versioned master reports that describe the complete state of the project at a given point in time.

The skill should produce a reliable reference document for project continuity and strategic visibility.

---

# RESPONSIBILITIES

- Generate master project reports
- Consolidate project state
- Consolidate project evolution
- Consolidate architecture status
- Consolidate roadmap status
- Consolidate ecosystem status
- Consolidate technical decisions
- Consolidate risks and priorities
- Produce versioned documentation

---

# NON-GOALS

- Do not invent project information
- Do not rewrite project history
- Do not replace detailed documentation
- Do not make strategic decisions

---

# TOOLS

- Read
- Write
- Glob
- Grep

---

# INPUTS

- Project-state reports
- Consolidated project knowledge
- Evolution reports

# WORKFLOW

1. Gather project-wide information
2. Consolidate current state
3. Consolidate historical evolution
4. Assess architecture and roadmap status
5. Identify priorities and risks
6. Generate master version document
